Project Description:

The Personal Portfolio Website is a fully responsive and interactive frontend project developed using core web technologies.  
The website is designed to professionally present my personal profile, technical skills, academic projects, and contact information in a clean and well-structured layout.

The main objective of this project is to demonstrate practical knowledge of modern web development concepts such as responsive design, structured layouts, smooth navigation, UI consistency, and JavaScript-based interactivity.

---

Technologies Used:

HTML5 – Semantic page structure and content  
CSS3 – Styling, layout design, animations, and responsiveness  
JavaScript – DOM manipulation, interactivity, navigation handling, and form validation  
EmailJS – Contact form message handling  

---

Implemented Features:

Responsive Navigation Bar with Mobile Toggle  
Sticky Header on Scroll  
Hero / Introduction Section with Typing Animation  
About Me Section with Profile Image  
Skills Section using Responsive Grid Layout  
Projects Section with Interactive Hover Overlay  
Live Project & GitHub Links Integration  
Contact Form with Client-side Validation and EmailJS Integration  
Smooth Scrolling Navigation  
Active Navbar Highlight on Scroll  
Scroll Reveal Animations  
Light/Dark Mode Toggle  
Hover Effects & UI Enhancements  
Fully Responsive Design using Media Queries  

---

Functionality Overview:

Smooth navigation between all sections  
Typing text animation for dynamic role display  
Scroll-based active navigation highlighting  
Interactive project cards with external links  
Form validation with real-time user feedback  
Email sending functionality using EmailJS  
Responsive layout for mobile, tablet, and desktop  
Theme switching using Light/Dark mode toggle

Challenges Faced:
